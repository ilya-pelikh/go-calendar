# Go Calendar — планировщик задач

## Описание проекта

Проект — простой веб‑планировщик задач с HTTP API и фронтендом. Сервер хранит задачи в SQLite, отдаёт статику из `web/` и предоставляет API для работы с задачами и расчёта следующей даты.

## Задания со звёздочкой выполнены 👍

## Запуск локально

1. Создайте `.env` в корне проекта (пример):

   ```
   TODO_PORT=7540
   TODO_ENV="development"
   TODO_DBFILE="scheduler.db"
   TODO_PASSWORD="privet"
   TODO_JWT_KEY="super_secret_key"
   ```

   Если `TODO_PASSWORD` задан, все `/api/*` (кроме `/api/nextdate`) требуют авторизацию (cookie `token`).

2. Запуск:

   ```
   go run ./cmd
   ```

3. Откройте в браузере:
   ```
   http://localhost:7540/
   ```

## Запуск тестов

Тесты обращаются к запущенному серверу. Запустите сервер в отдельном терминале, затем:

```
go test ./tests
```

## Docker (если используете)

Сборка:

```
docker build -t diploma .
```

Запуск:

```
docker run --rm -p 7540:7540 --env-file .env diploma
```

Фронтенд будет доступен по адресу:

```
http://localhost:7540/
```
